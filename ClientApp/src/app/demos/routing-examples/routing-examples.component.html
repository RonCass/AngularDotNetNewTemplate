Routing Examples

These examples can be seen in both the app.module.ts for module, and demos.routing.module.ts for most of the other examples

Module Routing
  • Look at how the Demo Module is used in the app.module routes
  • Example: {{ '{' }} path: 'demos', loadChildren: './demos/demos.module#DemosModule', canLoad: [AuthGuard] {{ '}' }}
  • "canLoad" calls the AuthGuard to authorize the user against this module. CanLoad will not even load the module if the AuthGuard returns false.
  • "canActivate" calls the AuthGuard to authorize the user against this module. The module will still be loaded but the user will not activate if AuthGurard returns false.
Default Routing
  • Example: {{ '{' }} path: '', component: DefaultRoutePageComponent, canActivate: [AuthGuard] {{ '}' }}
  • Example Redired to another path: {{ '{' }} path: '', redirectTo: '/heroes', pathMatch: 'full' {{ '}' }}
  • Links to pages in the demo module require a "demo/" before each link. But if a user just goes to http://localhost/demos/ they will get an empty page unless you point those to a page in the module. And it doesnt have to be its own page like I have it but any component in the module will do.
Basic Component Routing
  • Example: {{ '{' }} path: 'routingExamples', component: RoutingExamplesComponent, canActivate: [AuthGuard] {{ '}' }}
  • The "path" is what you will use in your links. So for this page, routingExamples, the link would be: <a [routerLink]='["/demos/routingExamples"]'>Routing Examples</a>. You should also see this URL in the browser since you are on this page now.
  • Other Examples:
    <a routerLink="/demos/routingExamples">Routing Examples</a> - Square brackets aren't required for simple paths.
    Route With ID or Token - <a [routerLink]='["/demos/routingWithToken:token"]'>Route With Token</a> - Route With Token Examples
    Or using route paramm array: <a [routerLink]='["/demos/routingWithToken:token", routeToken]'>Route With Token</a> - Route With Token Examples
  • Page Not Found Route: {{ '{' }} path: '**', component: PageNotFoundComponent {{ '}' }}
